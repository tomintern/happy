**happy** : A minimal [hapijs](http://hapijs.com/) API.

Hapijs Authentication with [JWT]() with [Boom](), [Joi](), [Good](), [Code]() and [Lab]()

## Usage

Clone a repository

```
git clone https://github.com/Phonbopit/happy.git
cd happy
npm install
```

Copy `.env.example` into `.env`

```
cp .env.example .env
```

Then, start a server

```
npm start

# Dev mode require nodemon
npm run dev

# Test
npm test

```
